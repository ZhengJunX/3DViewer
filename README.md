# 简介

本软件用于特定数据的3D建模，底层使用OpenGL，显示效果良好。经测试，一次性加载约12万点，其帧率最高仍然能达到测试显示器刷新率60Hz。

![](https://github.com/ZhengJunX/3DViewer/raw/master/resourse/image/3DViewer.png)

> 编译环境：Qt 5.12.0 MinGW 7.3.0 64bit、QGLViewer 2.7.1
